O que é Computação em Nuvem?

Computação em nuvem é um modelo em que recursos de tecnologia (como armazenamento, servidores, bancos de dados e aplicações) são oferecidos pela internet, sob demanda. Em vez de manter toda a infraestrutura localmente, você utiliza serviços hospedados em datacenters e paga apenas pelo que usar, com facilidade para aumentar ou reduzir capacidade conforme a necessidade.

Exemplos de serviços em nuvem

  • Armazenamento de arquivos (ex.: Google Drive, OneDrive, Dropbox)
  • Hospedagem de sites e aplicações (ex.: Vercel, Netlify)
  • Servidores virtuais (ex.: AWS EC2, Azure Virtual Machines)
  • Bancos de dados gerenciados (ex.: Amazon RDS, Cloud SQL)
  • Backup e recuperação de dados
